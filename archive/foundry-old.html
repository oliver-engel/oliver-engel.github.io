Product designer & programmer
After Effects Sketch Python

About

I wrote a Python program that takes font files and generatively manipulates them at the point level, the smallest building block of each letter. I then designed a UI through which designers and artists could use the program.

Concept

Inspired by Joel Swanson's article, "No-one Starts From Scratch: Type Design and the Logic of the Fork", I set out to create a program that would help me fork existing fonts into distorted versions of their former selves.

Enter FontForge, an open-source font editor that embeds Python.

Program development in IDLE

Flow

Users should be able to quickly and easily fork fonts into novel forms to use with their next project. The flow prioritizes easy uploading and effect stacking, much like the way that Instagram handles image manipulation.
