I set out to create a thorough and original product proposal for a font generation app, based off of my own Python program.

About

For a school project, I had previously written a Python command-line tool that let me manipulate fonts by altering the arrangement and parameters of their bezier curves and control points.

To take the project a step further, I later designed a full UI based off of that code, focusing on a mix of interaction design and visual design. I pursued the project to practice building a product from the ground up, and within the technical constraints set by my own program.
