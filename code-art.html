Code & Art

01. Splitscreen

Python program that uses the Newspaper library to scrape headlines from politically disparate outlets, and uses simple language analysis to combine them into a single headline. The output is displayed via ASCII animation, meant to mix aesthetics between digital and physical news media.

02. Endless

Unity 3D project coded in C#. Endless is a looping game that makes players complete a series of puzzles to advance to the next stage. When the cycle of stages, which roughly follows a mundane, 9-to-5 workday, is completed, the player begins the cycle again. But each time they restart the cycle, their perspective is warped, and increasing layers of harsh sound activate around them. Players try to get a high score of cycles completed.

03. Organspieler

Organspieler is a live video manipulation program written in Java for Processing. It uses a combination of audio monitoring and OpenCV to allow users to use their own bodies and found objects and direct an abstract music video in realtime. View the project on Github.
